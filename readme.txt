== Description ==
This plugin allows you to edit colors for block editor registered in theme, default block editor colors, and also you can create your own colors.

== Installation ==

1. Upload the plugin to the /wp-content/plugins/ directory.
2. Activate the Getwid WordPress plugin through the 'Plugins' menu in WordPress.
3. Use Getwid on your next page or post.
